# Password Analyzer

**Password Analyzer** is a Python toy by cntctchm that actually cares if your password sucks. It breaks it down, rates it, and whispers tips in your terminal.

---

## Features

- ✅ Checks password length & complexity
- ✅ Rates overall strength (weak / meh / solid)
- ✅ Flags common passwords (don’t be lazy)
- ✅ Suggests ways to get stronger
- ✅ Colorful terminal interface (thanks, Rich)
- ✅ Simple web UI with Flask
